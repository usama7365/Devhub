## üöÄ Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/usama7365/Devhub.git
cd devhub
```

2. Install dependencies:
```bash
npm install
```


3. Start the development server:
```bash
npm run dev
```
